## What Is CSS? CSS (Cascading Style Sheets) are the **style** of your document. They help control the look and feel of the document. You create **selectors** that target **HTML Elements** and then use **CSS Rules** to change the look and feel of your document. Selectors like: p { /* CSS Rules Here */ } Which targets all paragraphs, or: .rotator-slide { /* CSS Rules Here */ } Which targets *any* element with class="rotator-slide", or: #rotator { /* CSS Rules Here */ } Which targets *any* element with id="rotator".

CSS Rules go inside the {curly braces} that follow the **CSS Selector** and they look like this: 

 p {
	font-size: 2em;
	color: red;
}

Which will apply the font size of 2em and color of red to all paragraphs.

## What Is JS? JS (JavaScript) controls the **behavior** of your document. It allows you to do things such as animate items, hide or show items, react to user input and even process data and perform other general programming tasks. JS can look like this: var myVariable = "Hello World"; That line will create a variable called **myVariable** and assign it a value of **Hello World** (without the quotes). The structure and syntax of the language of JavaScript is much more complex than either HTML or CSS. This is because neither HTML nor CSS are full fledged programming languages. Neither of them have **variables**, **functions**, **objects** or **control structures** such as **if** statements or **loops**. JavaScript has *all* of these.

# Variables ## What *is* a variable? A variable is a storage location and an associated symbolic name (an identifier) which contains some known or unknown quantity or information, a value. ### In *English* professor. At its most basic we make a variable when we give a **name** to a **value**. So when we say: var myVariable = "Hello World"; We are taking the value ```Hello World``` and giving it the name ```myVariable```.
## Variable Types Variable *values* can be one of several different **types**. In this case, a value with text in it is called a **string** value. var myVariable = "Hello World"; Variables can also contain numbers, var meaningOfLife = 42; // Whole numbers are called 'integers' Variables that contain true / false values are called **booleans** var isJavaScriptCool = true; var isJavaScriptLame = false;
## Variable Types Variable can even hold lists of values (the list values themselves being **types** like strings or booleans) in what is called an **array** var myFavoriteThings = [ "Raindrops On Roses", "Whiskers On Kittens", ]; We will not go into an exhaustive list here but it is important that you understand that variables hold values and the values themselves have types.

Using a variable

The first time we create a variable we do so by using the JavaScript keyword var

var myVariable = "Hello World";

That line of code will create a new variable called myVariable and assign it the string value of Hello World

// Let's break this down.
	var myVariable = "Hello World";

	 |       |     └ The 'assignment' opperator - A single equals sign.

	 └ The 'var' keyword.
*/

The var keyword is only needed when you first create the variable. You do not need to use the var keyword again later when referencing or manipulating the variable.

Using a variable

# JavaScript Functions if a variable is a **name** for a **value**, then a function is a **name** for a set of commands or instructions. For example: ```javascript function sayHello(){ /* There can be many commands here in between the two {curly braces} Let's make the first command an alert to the user. */ alert("Hello World"); } ``` This creates a new **function** called "sayHello" which, when invoked, will execute the **alert** function that comes built into javascipt. The **alert** function is a **Built In Function** and the **sayHello** function is a **User Defined Function**. The only difference between them is that you don't have to write a built in function, it comes for free as a part of JavaScript.

# Using jQuery and the jQuery(); function The basic idea of jQuery is that you target some item on the page and then add or change the behavior of that item. For instance you could take a dropdown button in a menu and set it so that when it is hovered over, a new menu slides down from the top. That might look like this: ```javascript jQuery(".nav button.view-menu").hover(function(){ jQuery(".nav .more-menu").slideDown(); }, function(){ jQuery(".nav .more-menu").slideUp(); }); ``` DEMO HERE
# Using jQuery and the jQuery(); function You could also make it so that when clicking on a button to dismiss a popup, you remove an element from the document. ```javascript jQuery(".dialog .dismiss").click(function(){ jQuery(this).parent().remove(); }); ``` DEMO HERE These are just simple examples. With jQuery and a little imagination the possibilities are endless.
# How to select an item with jQuery(); ## (So you can alter its behavior) the ```jQuery();``` function takes a single parameter: A string containing a CSS style selector indicating what item, or items, on the page to select. Some examples: ```javascript jQuery("ul li.even"); /* The line above selects all list items, with a class of "even" in an any unordered list. */ jQuery("#rotator .slide:first"); /* The line above selects the first item with a class of "slide" inside the item with an id of "rotator". */ ```
# How to select an item with jQuery(); ## (So you can alter its behavior) The HTML items or your page that match your selector are added into something called the **wrapped set**. This is a special grouping of items that jQuery uses to know what items to alter or add behavior to. When you run a jQuery **method** the items in the **wrapped set** are the ones that will be altered or effected.

jQuery Methods

the jQuery library comes with many many MANY different methods. We won't discuss all of them here but if you want to see them all, and how to use them, you can go to api.jquery.com.

We will discuss a few methods here and a few more once we start making our rotator.

  • the .hide(); method will hide all elements in the wrapped set.
  • the .show(); method will show all elements in the wrapped set.
  • the .css(); method allows you to alter the look of the elements in the wrapped set by passing two parameters. The CSS property you want to edit and the new value of that property
# jQuery Methods For example: ```javascript jQuery(".alert").css("color", "red"); ``` Will take elements with a class of alert and make the text in them red. This by no means an exhaustive list, we will take notice when we intruduce new methods as we go on.

# One last thing before we get into it the rotator. ## Timing. Your webpage, like all webpages, gets downloaded by the browser. The browser downloads the page from top to bottom and when it see any javascript, it runs it. Right then and there. So if you add this: ```javascript $("p.highlight").css("background-color", "yellow"); ``` and then later in the page you put ```html <p class="highlight">This is an important highlighted paragraph</p> ``` The javascript code will run BEFORE the browser will even see the paragraph tag. That means that jQuery won't match the element and won't make it's background color yellow!
# Enter Document Ready Document ready is an **event**. It's a *point in time* when the browser has downloaded enough of your page that it has seen all your tags and it can find them based off of the CSS selectors that you tell jQuery. Running that same code on the **document ready** event looks like this: ```javascript $(document).ready(function(){ $("p.highlight").css("background-color", "yellow"); }); ``` Document Ready is a little bit of a complex idea and again because we only have a small amount of time we won't explain how it works in great detail here. # Finally... ## We get to the image rotator So, What IS an image rotator? At it's most basic, a rotator is a list of images that uses javascript to show one image on its own, then another, then another... SO if we used HTML like this: ```html <div id="rotator"> <ul class="slides"> <li><img src="content/rotator/image1.jpg" alt="First Slide"></li> <li><img src="content/rotator/image2.jpg" alt="Second Slide"></li> <li><img src="content/rotator/image3.jpg" alt="Third Slide"></li> </ul> </div> ``` The javascript would show the first and hide the rest. Then, after a couple of seconds, show the second and hide the rest. Then keep going until all of the slides have been shown. Then start again at the beginning.
# How Would We Build That? In order to build something like this we need to know how to hide objects, how to show them and how to wait a set amount of time before moving to the next image. Using the HTML from before we could do something like this: ```javascript setInterval(function(){ if($('#rotator .rotator-items li:visible').next().length == 0){ $('#rotator .rotator-items li:visible').hide(); $('#rotator .rotator-items li:first').show(); } else { $('#rotator .rotator-items li:visible').hide().next().show(); } }, 2000); ``` You can see an example of this in the next slide
