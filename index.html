## What Is CSS? CSS (Cascading Style Sheets) are the **style** of your document. They help control the look and feel of the document. You create **selectors** that target **HTML Elements** and then use **CSS Rules** to change the look and feel of your document. Selectors like: p { /* CSS Rules Here */ } Which targets all paragraphs, or: .rotator-slide { /* CSS Rules Here */ } Which targets *any* element with class="rotator-slide", or: #rotator { /* CSS Rules Here */ } Which targets *any* element with id="rotator".

CSS Rules go inside the {curly brackets} that follow the **CSS Selector** and they look like this: 

 p {

	color: red;

## What Is JS? JS (JavaScript) controls the **behavior** of your document. The structure and syntax of the language of JavaScript is much more complex than either HTML or CSS. This is because neither HTML nor CSS are full fledged programming languages. # Variables ## What *is* a variable? A variable is a named memory location which contains some known or unknown quantity or information. ### In *English* professor. At its most basic we make a variable when we give a **name** to a **value**. So when we say: var myVariable = "Hello World"; We are taking the value ```Hello World``` and giving it the name ```myVariable```.
## Variable Types Variable **values** can be one of several different **types**. The most basic is a **number** var meaningOfLife = 42; // Whole numbers are called 'integers' Javascript can do math with numbers. ```javascript var endingValue = 5 * 2; // the asterisk '*' is the symbol for multiplication. // This will set the value of 'endingValue' to 10. ``` The calculations work the same whether the values are either numbers, or values stored in a variable: ```javascript var startingValue = 5; var multiplier = 2; var endingValue = startingValue * multiplier; // 'endingValue' is still 10. ```
## Variable Types When you store any amount of text, like words or sentances for example, the variable type is a **string** value. ```javascript var myVariable = "Hello World"; ``` You can add strings together to make longer strings ```javascript var first = "Hello"; var second = "World"; var wholePhrase = first + " " + second; ``` The javascript above adds **three** items together: 1. the variable **first** with the value "Hello" * the **anonymous** string value " " * the variable **second** with the value "World" "Hello" + " " + "World" = "Hello World"
## Variable Types Javascript understands stings as values that are wrapped in either 'single quotes' or "double quotes". You can use either style of quotation mark but you must both start and end the string with the same mark you began with: ```javascript var works = "This string value's the bomb!!!"; var fails = 'This string value's the bomb!!!'; '; var alsoWorks = "Then I said: 'That string value was the bomb!!!'."; var alsoFails = "Then I said: "That string value was the bomb!!!"."; /* ' /* ^ ^ | | |_____________| | Javascript thinks the string ends with the apostrophe. It is then expecting the command to end with a semicolon (;) This doesn't happen and the following Error is thrown: "SyntaxError: missing ; before statement" */ ```
## Variable Types You can use the double quote mark (") inside a string that is **delimited** by double quotes by putting a backslash (\\) in front of it. ```javascript var totallyWorks = "Then I said: \"No way you can do that!!!\"."; /* ' /* ^ ^ | | |___________________________________________| | Javascript sees a string starting and ending with the same symbol. No Error is thrown. The variable totallyWorks is assigned a string value of: Then I said: "No way you can do that!!!". */ ``` We use the quote marks to **delimit** a string but that does not add the **value** of the quote mark to the **content** of the string.

# JavaScript Functions if a variable is a **name** for a **value**, then a function is a **name** for a set of commands or instructions. For example: ```javascript function sayHello(){ /* There can be many commands here in between the two {curly brackets} Let's make the first command an alert to the user. */ alert("Hello World"); } ``` This creates a new **function** called "sayHello" which, when invoked, will execute the **alert** function that comes built into javascipt. The **alert** function is a **Built In Function** and the **sayHello** function is a **User Defined Function**. The only difference between them is that you don't have to write a built in function, it comes for free as a part of JavaScript.

# Using jQuery and the jQuery(); function The basic idea of jQuery is that you target some item on the page and then add or change the behavior of that item. For instance you could take a dropdown button in a menu and set it so that when it is hovered over, a new menu slides down from the top. That might look like this: ```javascript jQuery(".nav button.view-menu").hover(function(){ jQuery(".nav .more-menu").slideDown(); }, function(){ jQuery(".nav .more-menu").slideUp(); }); ``` DEMO HERE
# Using jQuery and the jQuery(); function You could also make it so that when clicking on a button to dismiss a popup, you remove an element from the document. ```javascript jQuery(".dialog .dismiss").click(function(){ jQuery(this).parent().remove(); }); ``` DEMO HERE These are just simple examples. With jQuery and a little imagination the possibilities are endless.
# How to select an item with jQuery(); ## (So you can alter its behavior) the ```jQuery();``` function takes a single parameter: A string containing a CSS style selector indicating what item, or items, on the page to select. Some examples: ```javascript jQuery("ul li.even"); /* The line above selects all list items, with a class of "even" in an any unordered list. */ jQuery("#rotator .slide:first"); /* The line above selects the first item with a class of "slide" inside the item with an id of "rotator". */ ```
# How to select an item with jQuery(); ## (So you can alter its behavior) The HTML items or your page that match your selector are added into something called the **wrapped set**. This is a special grouping of items that jQuery uses to know what items to alter or add behavior to. When you run a jQuery **method** the items in the **wrapped set** are the ones that will be altered or effected.

jQuery Methods

# One last thing before we get into it the rotator. ## Timing. Your webpage, like all webpages, gets downloaded by the browser. The browser downloads the page from top to bottom and when it see any javascript, it runs it. Right then and there. So if you add this: ```javascript $("p.highlight").css("background-color", "yellow"); ``` and then later in the page you put ```html <p class="highlight">This is an important highlighted paragraph</p> ``` The javascript code will run BEFORE the browser will even see the paragraph tag. That means that jQuery won't match the element and won't make it's background color yellow!

# Finally... ## We get to the image rotator So, What IS an image rotator? At it's most basic, a rotator is a list of images that uses javascript to show one image on its own, then another, then another... SO if we used HTML like this: ```html <div id="rotator"> <ul class="slides"> <li><img src="content/rotator/image1.jpg" alt="First Slide"></li> <li><img src="content/rotator/image2.jpg" alt="Second Slide"></li> <li><img src="content/rotator/image3.jpg" alt="Third Slide"></li> </ul> </div> ``` The javascript would show the first and hide the rest. Then, after a couple of seconds, show the second and hide the rest. Then keep going until all of the slides have been shown. Then start again at the beginning.

## Let's break that down ```javascript $('.flexslider').flexslider({ slideshowSpeed : 2000 }); // | | | || | | |└ End Parenthesis // | | | || | | └ End Curly Brackets // | | | || | └ The property has // | | | || | a VALUE of 2000 // | | | || └ Set the object PROPERTY // | | | || 'slideshowSpeed' // | | | |└ Curly brackets indicate a JavaScript OBJECT // | | | └ Parenthesis indicate a Function Call // | | └ The Flexslider Method // | └ The CSS selector '.flexslider' will add any items // | with the class 'flexslider' to the jQuery wrapped set // └ The jQuery Function ``` We pass an **object** as a **parameter** to the flexslider **method** of jQuery. The **object** has **properties** and **values** that flexslider will use to override its defaults.
# Objects An **object** is a JavaScript **data type**, just like a **string** or a *number* is. ```javascript var myObject = { animation : "slide", slideshowSpeed : 2000 }; ``` You can assign **objects** to **variables** and then use them the same as other **variables**. ```javascript var myObject = { animation : "slide", slideshowSpeed : 2000 }; $('.flexslider').flexslider(myObject); ``` This creates a flexslider with a sliding animation every 2 seconds.
# I Can Haz Responsive? Flexslider comes out of the box with the ability to be responsive. Try it now. resize your browser and watch your images scale as needed. ### Pro Tip: Make your images only as wide as they NEED to be. That is, the largest size that you will need in your design. They will automatically scale down as needed, but if you make them larger then they need to be the file size will be larger and your page will download slower.

# Clickable Slides It is super simple to make your slides able to link to another page, or even another section on the same page. Let's look at the HTML we are using now: ```html <div id="rotator"> <ul class="slides"> <li><img src="content/rotator/image1.jpg" alt="First Slide"></li> <li><img src="content/rotator/image2.jpg" alt="Second Slide"></li> <li><img src="content/rotator/image3.jpg" alt="Third Slide"></li> </ul> </div> ``` Now let's just wrap those images in links using the ```<a>``` tag in **HTML** ```html <div id="rotator"> <ul class="slides"> <li><a href="#section1"><img src="content/rotator/image1.jpg" alt="First Slide"></a></li> <li><a href="#section2"><img src="content/rotator/image2.jpg" alt="Second Slide"></a></li> <li><a href="#section3"><img src="content/rotator/image3.jpg" alt="Third Slide"></a></li> </ul> </div> ```
